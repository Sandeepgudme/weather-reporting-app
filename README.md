Weather Application

A simple web-based weather application that displays current weather conditions and forecasts for up to 2 days ahead.

Features


Displays current weather conditions for a given city or user's location
Shows forecast for up to 4 days ahead
Supports searching by city name
Includes option to use device location for weather lookup
Responsive design for mobile and desktop viewing
Error handling and user feedback for various scenarios
Getting Started
These instructions will guide you through setting up the project on your local machine for development and testing purposes.

Prerequisites

Modern web browser (Chrome, Firefox, Safari, Edge)
Internet connection
Basic knowledge of HTML, CSS, tailwind css and JavaScript

Installation
Clone the repository:
git clone https://github.com/sandeepgudme/weather-app.git

Navigate to the project directory:
cd weather-app

Open the index.html file in your preferred web browser.
Usage

Enter a city name in the search bar and click "Search".
Alternatively, click "Use Location" to get weather information based on your device's location.

View current weather conditions and forecast for the next 2 days.

API Documentation
The application uses the WeatherAPI.com service. For details on the API endpoints and parameters used, please refer to the WeatherAPI documentation.
